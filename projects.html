Music Platform Website

The Kadoma Billboard Chart System is a simple project that is inspired by the Apple Web Application. This system requires system credentials to access all of the data and functionalities of the kadoma billboard chart. The system has 2 types of users which is the Administrator and the Subscriber. The Admin user can manage all the functionalities of the system including the music uploaded.

The Subscriber user can only manage the songs they uploaded and album they created but since this also like and music publishing site, the system allows all users to listen all uploaded music or songs and also listen to the album songs created by other subscribers and admin. The Subscriber user can also download any song that can be accessed in this system.

The Vote Page is the page where different users can view and vote for their favourite album. The system will automatically add votes to that artist and the top 6 artists with more votes will be displayed on the home page.
